# win11_refresh_icon

윈도우 11에서 아이콘 캐시를 삭제하고 탐색기(Explorer)를 재시작해서 **아이콘/썸네일 표시가 꼬였을 때** 빠르게 새로고침(리프레시)하는 간단한 배치 파일입니다.

## 포함 파일

- `refresh.bat` : 아이콘 캐시(`iconcache_*.db`)를 삭제하고 `explorer.exe`를 재시작합니다.

## 사용 방법

1. `refresh.bat`을 다운로드합니다.
2. **관리자 권한으로 실행**을 권장합니다.
3. 잠시 화면이 깜빡이거나 작업 표시줄/탐색기가 재시작됩니다.

## 동작 원리

`C:\Users\%USERNAME%\AppData\Local\Microsoft\Windows\Explorer` 폴더로 이동한 뒤, 숨김 속성의 아이콘 캐시 파일을 보이도록 설정하고(`attrib -h`), `iconcache_*.db` 파일을 삭제한 다음 `explorer.exe`를 강제 종료 후 다시 시작합니다.

## 주의사항

- 실행 중 **탐색기(Explorer)가 강제로 종료**되므로, 작업 중인 파일 탐색기 창이 닫힐 수 있습니다.
- 아이콘/썸네일 캐시가 재생성되는 동안 순간적으로 로딩이 발생할 수 있습니다.
- 회사/학교 PC 등 정책이 있는 환경에서는 실행이 제한될 수 있습니다.
